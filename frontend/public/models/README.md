# 3D Models

## How to add the Sketchfab drone model:

1. Go to: https://sketchfab.com/3d-models/drone-dji-mini-2-73a8c297b75e4686ae40fd44c0b96a55
2. Click "Download 3D Model" (requires free Sketchfab account)
3. Download in GLTF format (.glb or .gltf)
4. Place the downloaded file here as `drone.glb`
5. The BallPit component will automatically use it

## Alternative:
If the model isn't available for download, you can use any other drone model in GLTF/GLB format.
Free drone models can be found on:
- Sketchfab (filter by "Downloadable")
- Poly Pizza
- Free3D
- TurboSquid (free section)
